# Cranberry — Social Performance Dashboard

A self-contained, single-file social media performance dashboard for **Cranberry USA**, covering Instagram, Facebook, and LinkedIn for the May 2026 reporting period.

## What's inside

- **Instagram tab** — @cranberry.usa · Apr 26 → May 25, 2026 · 1,675 followers
- **Facebook tab** — Cranberry · Apr 26 → May 24, 2026 · 6,027 followers
- **LinkedIn tab** — Cranberry USA · Apr 1 → May 24, 2026 · 20.2% avg engagement rate

Built around the cross-channel story of the CDA 2026 "Cran Land" activation and the @the_guygienist creator collab — both surfaced as top performers on every channel.

## Tech

- Single static HTML file, no build step, no dependencies
- Embedded base64 logo, embedded SVG charts and icons
- Loads Barlow + Barlow Condensed from Google Fonts
- Fully responsive (desktop / tablet / mobile breakpoints)
- Brand-accurate colors sampled from the official Cranberry mark (`#EE1C25` red, `#00081B` navy)

## Local preview

Open `index.html` in any modern browser. No server required.
